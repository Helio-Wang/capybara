Output visualization¶

For visualizating reconciliations (parasite tree drawn on top of the host tree), use the original viewer for .

For visualizating event partitions or equivalence classes (colorings of the parasite tree with animation), use the new animated web tool Quokka Viewer.

The Quokka visualizer tool takes two input formats generated from the third tab Convert enumeration files for visualization of the main Quokka tool:

  • File input (DOT format) can be saved into the filesystem by clicking the Save button.

  • Plain text input can be copied into the user’s clipboard by clicing the Copy button.

Note

When the app closes, the clipboard content will also be cleared.
